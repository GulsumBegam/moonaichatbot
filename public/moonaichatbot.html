Capabilities

Everything you need

A complete full-stack AI application from frontend to database, in one cohesive cosmic experience.

🤖
Real-time AI Responses
Powered by OpenAI GPT-4o-mini via custom REST API routes. Full conversation context sent with each request.
💾
Persistent Chat History
Prisma ORM with SQLite stores all conversations and messages. Load any past chat instantly from the sidebar.

🗂️
Multi-Conversation
Create unlimited conversations, switch between them, rename, and delete with smooth animations.

Backend

REST API Routes

Full backend built with Next.js App Router API routes. Clean, typed, and documented.

POST /api/chat Send message → get AI response + save to DB
GET /api/history Fetch all conversations (sorted by latest)
GET /api/history/:id Get conversation with full message history
DELETE /api/history/:id Delete conversation and all its messages
PATCH /api/history/:id Update conversation title

Get Started

Quick Setup

From zero to running Moon AI in under 2 minutes.

1
Clone & Install
git clone https://github.com/YOUR_USERNAME/moon-ai-chatbot.git && cd moon-ai-chatbot && npm install
2
Configure Environment
Copy .env.example → .env.local and add your OpenAI API key from platform.openai.com
3
Setup Database
npx prisma generate && npx prisma db push
4
Launch 🚀
npm run dev
Open http://localhost:3000 and start chatting with the cosmos.
